# 仓库描述

**小红书爬虫**，**抖音爬虫**， **快手爬虫**， **B站爬虫**...。  
目前能根据id下载B站用户所有视频、爬取抖音/快手用户视频信息。

原理：利用Selenium模拟浏览器操作，需要手动登录。

## 使用方法(请自行安装依赖项)

### 进入根目录
   ```shell   
   # 进入项目根目录
   cd MultiCrawler
   
   ```
   
### 运行爬虫程序

   ```shell
   
   python3 get_user_video_list.py
   # 浏览器弹出后在40秒内完成登录    
   ```


### 数据保存
- 支持保存到关系型数据库（sqlite3）
